# Part 4 - Student Performance Analysis & Prediction

## Description
Analyzes student performance data, creates visualizations,
and builds a machine learning model to predict pass/fail.

## Tasks
- Task 1: Data Exploration with Pandas
- Task 2: Matplotlib Visualizations (5 plots)
- Task 3: Seaborn Visualizations (2 plots)
- Task 4: Machine Learning with scikit-learn

## Files Included
- students.csv
- plot1_bar.png to plot7_seaborn_scatter.png

## How to Run
```bash
jupyter notebook part4_visualization_ml.ipynb
```

## Concepts Used
Pandas, Matplotlib, Seaborn, Logistic Regression, StandardScaler
